Утилита управления виртуальной ФС

утилита vfs.exe позволяет 
- копировать файлы на виртуальную ФС;
- читать файлы с виртуальной ФС;
- удлять файлы на виртуальной ФС;
- просматривать содержимое корневого каталога виртуальной ФС;
- загружать образ драйвера NT и выполнять его;


Утилита поддерживает следующие команды:
DIR -  просмотреть содержимое каталога (пример: vfs dir)
COPY - скопировать файл на ФС или с ФС (пример: vfs copy c:\windows\calc.exe vfs\calc.exe
						vfs copy vfs\calc.exe c:\test\calc.exe)
DEL -  удалить файл (пример: vfs del vfs\calc.exe)
LOAD - загрузить указанный драйвер (пример: vfs load c:\test\sample.sys
					    vfs load vfs\sample.sys )

(Для адресации файлов, находящихся на виртуальной ФС, используется префикс "vfs\")
